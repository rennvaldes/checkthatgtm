## Border Rules

### Column 1 (Icon + Title):

- `border-t` - Only first step
- `border-b` - All steps
- `border-l` - All steps
- `border-r` - All steps

### Column 2 (Description):

- `border-t` - Only first step
- `border-b` - All steps
- `border-r` - All steps

### Mobile:

- Columns stack vertically
- Icon+Title keeps all 4 borders (t/b/l/r)
- Description appears below without top border
